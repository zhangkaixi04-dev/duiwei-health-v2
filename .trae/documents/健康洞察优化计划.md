## 📋 规划概述

本次优化在之前确认版本的基础上进行，主要改进：
1. 睡眠达标标准改为7-8小时（科学依据：全因死亡率最低）
2. 简化标题和布局优化
3. 增强AI总结逻辑（结合体质）
4. 新增左右滑动数据模块

---

## 一、睡眠达标标准修改

### 1.1 修改内容
- **原标准**：7-9小时 + 23点前入睡
- **新标准**：7-8小时（科学依据：全因死亡率最低）

### 1.2 科学依据
- 睡眠时间7-8小时的人群全因死亡率最低
- 少于7小时或超过8小时都会增加死亡风险

## 三、AI总结逻辑增强

### 3.1 总结结构
1. 整体评价
2. 分模块评价（饮食、睡眠、运动、排便）
3. 哪些对改善问题体质有帮助
4. 哪些要避免
5. 具体建议

### 3.2 结合体质的评价逻辑

| 体质类型 | 饮食建议 | 避免建议 |
|---------|---------|---------|
| **阳虚质** | 温补食物（羊肉、红枣、姜茶） | 生冷食物、冰饮 |
| **湿热质** | 清热祛湿（绿豆、冬瓜、金银花茶） | 辛辣、酒、油炸 |
| **痰湿质** | 清淡饮食、陈皮、普洱 | 肥甘厚味、甜食、奶茶 |
| **气虚质** | 甘温补气（鸡肉、山药） | 耗气食物（萝卜、山楂）、生冷 |
| **阴虚质** | 滋阴食物（银耳、百合、梨） | 辛辣、烧烤、羊肉、姜蒜 |
| **血瘀质** | 活血食物（山楂、醋） | 寒凉、冷饮、油腻 |
| **气郁质** | 疏肝食物（花茶、柑橘类） | 浓茶、咖啡、烈酒 |
| **特禀质** | 清淡、固表食物 | 发物（海鲜、鹅、笋、酒） |
| **平和质** | 均衡饮食 | 过量、油腻、高糖 |
